# TODO App - Beta Release

Welcome to the TODO App beta! This folder contains two versions of the application for Windows.

---

## 📦 Files Included

### 1. **TODO App Setup 0.0.0.exe** (221 MB) - RECOMMENDED FOR MOST USERS
**Type:** Full Windows Installer (NSIS)

**Features:**
- Professional installation wizard
- Automatic updates support
- Creates Start Menu shortcuts
- Creates Desktop shortcut
- Proper add/remove programs entry
- Easy uninstall support
- Runs from Program Files

**Best for:** First-time installation, distribution to beta testers

---

### 2. **TODO App.exe** (217 MB) - Portable Version
**Type:** Standalone Portable Executable

**Features:**
- No installation required
- Can run from any location (Desktop, USB, folder)
- No registry modifications
- Can be deleted without cleanup issues
- Ideal for quick testing

**Best for:** Quick testing, USB portability, minimal setup

---

## 🚀 Installation Instructions

### Option A: Full Installation (Recommended)

1. **Locate the file:** `TODO App Setup 0.0.0.exe` on your Desktop
2. **Double-click** the installer
3. **Follow the setup wizard:**
   - Click "Next" to proceed
   - Choose installation directory (default is recommended)
   - Select additional options (Desktop shortcut, Start Menu shortcut)
   - Click "Install"
4. **Launch the app:**
   - From the Desktop shortcut, or
   - From Start Menu → TODO App

---

### Option B: Portable Installation (No Setup Required)

1. **Locate the file:** `TODO App.exe` on your Desktop
2. **Double-click** to run immediately - no installation needed!
3. **The app will launch** in a new window

---
